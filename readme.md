Structure of folder:-

│

│   ├── index.html  (Main UI using Tailwind CSS)  
│   ├── script.js  (Handles API calls & UI interactions)  
│   ├── styles.css  (Optional, but Tailwind is used instead)  
│  
│── README.md  (Instructions for setup & usage)
